The display Property

display: none:

This is the none display. By using this, the element is completely removed. Neat huh?

display: inline:

Displays an element as an inline element (like in a span tag). Any height and width properties will have no effect Neat huh?

display: block:

Displays an element as a block element (like the p tag). It starts on a new line, and takes up the whole width Neat huh?

display: inline-block:

Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values Neat huh?

Position Property

position: static;

An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page:

position: relative;

An element with position: relative; is positioned relative to its normal position:

position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed):

position: fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:

Floats

Without clear

div1
div2 - HEY LISTEN! div2 is after div1 in the HTML code. Since div1 floats to the LEFT, the text in div2 flows AROUND div1.


With clear

div3
div4 - Beep-Boop-Beep-Boop! clear: left; moves div4 DOWN below the floating div3. The value "left" CLEARS elements floated to the LEFT. You can also clear "right" and "both".
